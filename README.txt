Deskripsi singkat
Sistem berbasis smarthome untuk mengatur perabot rumah secara mudah dan efisien. Sistem dapat mematikan dan mematikan A/C melalui perangkat android dan menyalakan lampu secara otomatis
saat pengguna berpindah tingkat rumah. Sistem dapat memberikan notifikasi kepada pengguna lain didalam rumah saat ada pengguna yang sampai di rumah.

Panduan instalasi:
Install aplikasinya
Setup arduino kepada raspberry

Cara pengguna:
Geser kiri-kanan untuk berpindah fungsionalitas
Untuk menyalakan alarm tekan tombol bergambar jam untuk menambahkan waktu alarm
Untuk mengaktifkan fitur menyalakan lampu otomatis saat berpindah tingkat, geser ke laman settings  lalu tekan tombol  bertulisan “This is ground floor”
Untuk menyala matikan lampu dan A/C geser ke laman “home iot” lalu tekan bagian yang ingin di ubah
